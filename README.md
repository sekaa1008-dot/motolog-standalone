# 적자생존

오토바이 정비소용 작업 일지 웹앱입니다.

## 기능

- 정비 기록 작성, 수정, 삭제
- 일일 마감 요약, 결제방식별 매출, 부품별 소진 수량
- 차량번호 뒤 4자리, 날짜, 작업내용 검색
- 작업 템플릿 빠른 입력
- 견적서 기반 기본 템플릿 자동 추가
- 부품 재고 등록, 보관 위치, 적용 모델, 경고 수량 관리
- 정비 기록 저장 시 사용 부품 자동 차감
- CSV 내보내기
- 카메라 촬영 및 갤러리 사진 첨부
- 차량 번호 직접 입력
- 차량 번호 입력 시 이전 작업 이력 표시
- 휴대폰 홈 화면 설치(PWA)

## 배포

정적 웹앱이라 빌드 과정이 필요 없습니다.

Netlify에서 GitHub 저장소를 연결할 때:

- Build command: 비워둠
- Publish directory: `.`

## 데이터 저장

서버 없이 브라우저 `localStorage`에 저장됩니다.
기기 간 데이터 동기화는 아직 포함되어 있지 않습니다.
